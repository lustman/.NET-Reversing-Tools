	             ver.0.1b


	- copy all files to plugins directory

You can run any exe , just edit file "Run_exe.ini"  and enter name.exe
if file located in plugins directory or full path and name if file is placed on disk.
 example : "XNResourceEditor.exe"  or  "C:\Program Files\PE Explorer\pexplorer.exe"
