VistaRegistryExample
    This example demonstrates how to store short license keys
    for all users in the registry, which requires administrative
    privileges

How to compile:

- Use the LIB tool (included with Visual C++) to create the library files:
  lib /def:dummy.def
  or
  lib /def:keygen.def
  and copy the resulting .lib files to the application directory
  The .def files are located in the SDK\C folder
